TITLE:Installation Instructions for Judge Eval System
AUTHORS:Maor Bernstein,Tian Zhang, and Wilson Burchenal

1. Copy all the files to the server
2. In terminal, first run construct_tar.sh
3. After finished, copy judgeEval.tar.gz and install.sh to the root directory (For ECC, it should be /webpages/{username}/) of the website
4. In the website's root directory, run install.sh
5. Delete install.sh
6. After finished, you can visit the admin portal at http://{url to your website}/admin.html and the judge interface at http://{url to your website}/eval.html

You can find sample projects for import under Resources/project_examples.csv
